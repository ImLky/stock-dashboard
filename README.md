# 주식 모니터링 대시보드

## 프로젝트 소개

주식 투자자를 위한 실시간 모니터링 대시보드입니다. 코스피, 코스닥, 원/달러, 비트코인 등 주요 지표와 개별 종목의 차트, 등락률 히트맵, 뉴스를 한 화면에서 확인할 수 있습니다.

> 본 프로젝트는 **포트폴리오 목적으로 구현**되었습니다. 실제 투자 판단에 사용하지 마세요.

## 주요 기능

- 실시간 주요 지수 (코스피, 코스닥, 원/달러, 비트코인)
- 개별 종목 차트 (Yahoo Finance 실제 데이터)
- 등락률 히트맵
- 종목 목록 (검색, 정렬)
- 뉴스 피드
- 다크 테마
- 반응형 (PC / Tablet / Mobile)

## 기술 스택

| 구분 | 기술 |
|------|------|
| 마크업 | HTML5 |
| 스타일 | CSS3 (Grid, 반응형) |
| 차트 | Chart.js |
| 데이터 | Yahoo Finance API |
| 폰트 | Pretendard |
| 배포 | GitHub Pages |
